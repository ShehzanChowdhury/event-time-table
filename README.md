# Event Table

A React-based event timetable application that displays events across multiple venues in a time-based grid layout. The application allows users to view events scheduled at different venues throughout the day, with support for date selection and interactive time slot visualization.

## Features

- Interactive event timetable with venue columns and time slot rows
- Date selection to view events for specific dates
- Responsive grid layout with customizable time intervals
- Material-UI components for a modern user interface

## Development

To run the application in development mode:

```bash
npm start
```

This will start the development server and open the app at [http://localhost:3000](http://localhost:3000). The page will reload automatically when you make changes.

## Build

To build the application for production:

```bash
npm run build
```

This creates an optimized production build in the `build` folder. The build is minified and ready for deployment.

## Available Scripts

- `npm start` - Runs the app in development mode
- `npm run build` - Builds the app for production
- `npm test` - Launches the test runner
- `npm run eject` - Ejects from Create React App (one-way operation)
